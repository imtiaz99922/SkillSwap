# SkillSwap - Client

Frontend scaffold (Vite + React + Tailwind) for SkillSwap authentication demo.

Quick start:

```powershell
cd "d:/CSE 471 Project/client"
npm install
npm run dev
```

Default dev server: http://localhost:5173
API base is `http://localhost:5000/api` by default. To change, set `VITE_API_BASE` in `.env`.
