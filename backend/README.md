# Backend (Express + Mongoose)

Quick setup:

1. Install dependencies

```powershell
cd "d:/CSE 471 Project/backend"
npm install
```

2. Start in development

```powershell
npm run dev
```

3. Health check

Open: http://localhost:5000/health

Notes:
- Put your real `MONGO_URI` in `.env` (already created using your provided URI).
